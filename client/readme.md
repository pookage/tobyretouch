# React Barebones

My kinda dev environment <3

## Core Libraries

- **Webpack** _v4.25.1_
- **Babel** _v7.1.5_
- **React** _v16.6.1_

## Features

- Webpack-dev-server for hot-reload
- `build-dev` and `build-prod` commands for development and deployment
- CDNs to keep bundle size down
- CSS modules

## Missing features to be added later
